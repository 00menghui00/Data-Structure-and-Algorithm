# 简单排序
- 思想：每轮遍历时找到最值，直接将最值放到目标位置。遍历完之后所有值已经按顺序排放。




```
void selection_sort(int arr[],int n){
    for (int i=0;i<n-1;i++){
        int max_index=i;
        for (int j=i+1;j<n;j++){
            if (arr[j]>arr[max_index]){
                max_index=j;
            }
        }
        if (max_index!=i){
            int tmp=arr[i];
            arr[i]=arr[max_index];
            arr[max_index]=tmp;
        }
    }
}
```
