
# 时间复杂度：

- 时间复杂度本质理解：**算法中的基本操作的执行次数，为算法的时间复杂度**。核心就是求解执行次数（循环的次数）

---


## 细节问题：
1. 表示时间和空间复杂度时，log表示以2为底的对数。
2. 时间复杂度为O(1)，并不是要求函数内部不能含有循环，而是要求循环的次数为常数次。


---

## 案例：
1. 斐波拉契函数时间复杂度分析
2. transformer中self-attention机制的时间、空间复杂度
